5- Explica brevemente los tipos de comandos de GIT. 
git init - inicializa el repositorio
git clone - se obtiene un repositorio de un servidor
git add - se agrega los cambios al index para poder commitear
git commit - se marca un punto de la historia
git pull - se obtiene los ultimos cambios del repositorio remoto
git push - se suben los ultimos cambios comiteados
git status - se obtiene el estado de los archivos/directorios
git merge - se combinan los cambios obtenidos con los creados
